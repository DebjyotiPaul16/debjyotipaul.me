## Stack

This site is built with:

- [Astro](https://astro.build) — static site framework
- [GSAP](https://gsap.com) — animations
- [Lenis](https://lenis.darkroom.engineering) — smooth scrolling
- Deployed on [Railway](https://railway.app)

